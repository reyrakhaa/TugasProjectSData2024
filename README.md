# Project Manajemen Inventori & Antrian Pembeli

Proyek ini adalah implementasi struktur data **Stack (LIFO)** dan **Queue (FIFO)** dalam pengelolaan inventori gudang serta sistem antrian pembeli menggunakan bahasa **C++**. Program ini memungkinkan admin untuk mengelola barang masuk, mencatat pembeli, dan menjual barang dengan aturan yang telah ditentukan.

## Fitur Utama:
- **Login Admin**: Admin dapat masuk menggunakan data KTP dan password.
- **Manajemen Gudang**:
  - Menambahkan barang ke gudang.
  - Menampilkan daftar barang.
  - Menampilkan daftar pembeli obat.
  - Menyerahkan obat ke pembeli.
- **Sistem Penjualan**:
  - **LIFO (Stack)**: Barang yang terakhir masuk akan dijual lebih dahulu.
  - **FIFO (Queue)**: Pembeli yang datang lebih awal akan dilayani lebih dahulu.
- **Sistem Pembeli**:
  - Pendaftaran pembeli menggunakan KTP.
  - Pembeli masuk dalam antrian berdasarkan waktu kedatangan.
  - Pembeli memesan obat menggunakan kode obat dan menyebutkan jumlah obat yang akan dibeli.

## Teknologi:
- **C++** : Proyek ini ditulis dalam bahasa pemrograman C++, menggunakan sistem LIFO (Stack) dan FIFO (Queue).
- **Dev C++ & Visual Studio Code**: Digunakan untuk mengembangkan dan menjalankan proyek ini. *Dev C++ & VSCode* memberikan lingkungan yang ideal untuk pengembangan *C++*.
